# 🌐 在线编辑指南

现在你可以**在浏览器中直接编辑笔记**，不需要本地操作！

## 🚀 访问管理后台

### 方法 1：从博客主页

访问你的博客：https://bwx-0910.github.io

点击顶部导航栏的 **"⚙️ 管理"** 按钮

### 方法 2：直接访问

https://bwx-0910.github.io/admin.html

## ⚙️ 首次配置

### 步骤 1：生成 GitHub Token

1. **访问** GitHub Token 页面：
   👉 https://github.com/settings/tokens/new?scopes=repo&description=Blog%20Editor

2. **填写信息**：
   - Note（描述）：`Blog Editor`
   - Expiration（过期时间）：选择 `No expiration`（永不过期）
   - **勾选权限**：`repo`（完整仓库访问权限）

3. **点击底部** "Generate token"

4. **复制 Token**（只显示一次，务必保存！）

### 步骤 2：配置管理后台

1. 打开 `admin.html`
2. 填写信息：
   - **GitHub 用户名**：`bwx-0910`（已预填）
   - **仓库名称**：`bwx-0910.github.io`（已预填）
   - **Personal Access Token**：粘贴刚才复制的 Token

3. 点击 **"保存配置"**

配置会保存在浏览器本地存储中，下次访问无需重新输入。

## ✏️ 在线编辑笔记

### 新建笔记

1. 点击 **"+ 新建笔记"**
2. 填写表单：
   - **标题**：如 `《原则》读书笔记`
   - **图标**：选择 emoji，如 📖 🧠 💰
   - **分类**：如 `管理`
   - **标签**：如 `管理, 决策, 原则`（逗号分隔）
   - **摘要**：100-200字简短摘要
   - **笔记内容**：完整的 Markdown 格式笔记

3. 点击 **"💾 保存笔记"**

### 编辑现有笔记

1. 在笔记列表中找到要编辑的笔记
2. 点击 **"编辑"** 按钮
3. 修改内容
4. 点击 **"💾 保存笔记"**

### 保存后

- ✅ 笔记会自动保存到 GitHub
- ✅ GitHub Pages 会自动构建（1-2 分钟）
- ✅ 刷新博客主页即可看到更新

## 🔐 安全说明

### Token 安全

- ✅ Token 只保存在你的浏览器本地
- ✅ 不会上传到任何服务器
- ✅ 只有你能访问

### 建议

1. **不要分享 Token** 给任何人
2. **定期更新 Token**
3. **不使用时可以删除** Token（删除浏览器缓存）

### 删除 Token

清除浏览器缓存：
- Chrome: `Ctrl + Shift + Delete`
- 选择"Cookie 和其他网站数据"
- 清除

## 🐛 常见问题

### Q: 保存后看不到更新？

**A**: 等待 1-2 分钟让 GitHub Pages 构建，然后强制刷新浏览器（Ctrl + F5）。

### Q: Token 失效了？

**A**: 
1. 重新生成 Token
2. 在 admin.html 中重新配置

### Q: 手机上能用吗？

**A**: 可以！在手机浏览器中访问 admin.html 即可。

### Q: 能否同时编辑？

**A**: 建议一次只在一个设备上编辑，避免冲突。

### Q: 本地和在线可以混用吗？

**A**: 可以！但要注意：
- 在线编辑后，本地需要先 `git pull` 拉取最新代码
- 或者删除本地 `.git` 文件夹重新 clone
