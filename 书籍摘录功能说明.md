# 📌 书籍摘录功能使用指南

## 功能特点

✅ **独立摘录管理** - 在编辑笔记时，可以单独管理书籍摘录  
✅ **一键添加** - 快速添加多条摘录，无需手动编辑 JSON  
✅ **随机颜色展示** - 每条摘录用不同的渐变色美观展示  
✅ **响应式布局** - 自动适配手机和电脑屏幕

## 使用方法

### 1. 在线编辑器添加摘录

1. 访问 `https://你的用户名.github.io/admin.html`
2. 编辑或新建笔记
3. 在"📌 书籍摘录"区域：
   - 在文本框中输入一条摘录
   - 点击"+ 添加摘录"按钮
   - 重复以上步骤添加更多摘录
4. 点击"删除"按钮可以移除不需要的摘录
5. 保存笔记

### 2. 本地编辑 Markdown 文件

在 `.md` 文件的 Front Matter 中添加 `quotes` 字段：

```yaml
---
title: 《少有人走的路》读书笔记
icon: 📖
date: 2026-01-16
category: 心理学
tags: 成长, 心理学, 自律
excerpt: 人生苦难重重，这是个伟大的真理...
quotes: ["人生苦难重重，这是个伟大的真理", "自律是解决人生问题的首要工具", "爱，是为了促进自我和他人心智成熟"]
---

# 正文内容...
```

## 展示效果

摘录会在笔记页面顶部以**卡片形式**展示：
- 🎨 每条摘录有独特的渐变色背景
- 📱 响应式网格布局
- ✨ 悬停时有动画效果
- 💬 引号装饰，突出"摘录"特性

## 渐变色方案

系统内置 10 种精美渐变色：
1. 紫色梦幻 (紫-粉)
2. 粉色浪漫 (粉-红)
3. 蓝色清新 (蓝-青)
4. 绿色活力 (绿-青)
5. 暖色阳光 (粉-黄)
6. 深蓝神秘 (青-紫)
7. 粉蓝柔和 (青-粉)
8. 橙色温暖 (橙-粉)
9. 樱花浪漫 (粉-紫)
10. 薰衣草梦境 (紫-蓝)

每条摘录按顺序循环使用这些颜色。

## 技术实现

- **存储格式**：JSON 数组存储在 Front Matter 中
- **渲染方式**：build.py 自动生成带样式的 HTML
- **颜色分配**：基于索引循环分配，确保一致性
- **样式定义**：CSS 渐变色类 `.quote-color-1` 到 `.quote-color-10`

## 注意事项

⚠️ **修改后记得重新构建**  
- 本地编辑后运行 `build.bat`
- 在线编辑会自动触发 GitHub Actions 构建

⚠️ **摘录长度建议**  
- 建议每条摘录 20-200 字
- 太长会影响卡片美观度

⚠️ **特殊字符处理**  
- 引号会自动转义
- 支持中英文混排

## 示例

### 好的摘录示例：
✅ "人生苦难重重，这是个伟大的真理"  
✅ "自律是解决人生问题的首要工具"  
✅ "爱，是为了促进自我和他人心智成熟"

### 不好的摘录示例：
❌ 太长：整段文字复制（建议拆分）  
❌ 太短：仅几个字（建议补充完整）  
❌ 无意义：重复的句子
