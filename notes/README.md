# 📝 笔记文件夹使用说明

这个文件夹存放你的读书笔记 Markdown 文件。

## 🚀 快速开始

### 1. 复制模板

复制 `笔记模板.md`，重命名为你的书名：

```
笔记模板.md  →  原则.md
```

### 2. 编辑内容

用任何文本编辑器打开，按模板填写内容。

推荐编辑器：
- **Typora** - 所见即所得，最适合写作
- **VS Code** - 功能强大
- **Obsidian** - 知识管理
- **记事本** - 简单直接

### 3. 注意事项

#### 顶部配置（重要！）

```markdown
---
title: 《书名》读书笔记    # 必填：笔记标题
icon: 📖                  # 必填：显示的图标
date: 2026-01-14          # 必填：日期格式 YYYY-MM-DD
category: 分类名称         # 必填：分类
tags: 标签1, 标签2        # 必填：标签（逗号分隔）
excerpt: 简短摘要...      # 必填：100-200字摘要
---
```

**必须保留顶部的 `---` 和 `---` 之间的配置！**

#### 常用图标

```

🧠 心理学
💰 理财
```

#### 分类建议

## ✍️ Markdown 语法

### 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 强调
```markdown
**粗体**
*斜体*
***粗斜体***
```

### 列表
```markdown
- 无序列表
- 无序列表

1. 有序列表
2. 有序列表
```

### 引用
```markdown
> 这是引用的话
```

### 代码
```markdown
`行内代码`
```

### 链接
```markdown
[链接文字](URL)
```

### 表格
```markdown
| 列1 | 列2 |
|-----|-----|
| 内容 | 内容 |
```

### 分割线
```markdown
---
```

## 🔄 更新到网站

### ✅ 自动转换（推荐）

**最简单**：双击 `publish.bat`（一键构建+部署）

**或分步执行**：
1. 写完 Markdown 文件
2. 双击 `build.bat`（转换为 data.js）
3. 双击 `deploy.bat`（部署到 GitHub）

### 工作原理

```
notes/*.md  →  build.bat  →  js/data.js  →  deploy.bat  →  GitHub Pages
```

**你只需要关心写 Markdown 文件，其他都是自动的！**

## 💡 写作建议

### 结构建议

1. **开头**：基本信息 + 核心观点（快速了解）
2. **中间**：详细笔记 + 精彩摘录（深入学习）
3. **结尾**：个人思考 + 实践应用（消化吸收）

### 长度建议

- **摘要**：100-200 字
- **完整笔记**：800-2000 字
- 太长可以分多篇

### 保存格式

**重要**：文件必须保存为 **UTF-8 编码**，否则中文会乱码！

在记事本中保存时，选择编码为 "UTF-8"。

## 🆘 常见问题

### Q: 可以用 Word 写吗？

A: 不建议。Word 格式不兼容。可以在 Word 写好后复制到 Markdown 文件。

### Q: 图片怎么添加？

A: 使用图床（如 SM.MS）上传图片，然后用链接：
```markdown
![图片描述](图片URL)
```

### Q: 可以插入视频吗？

A: 可以插入视频链接，但不能直接嵌入视频文件。

### Q: 数学公式怎么写？

A: 使用 LaTeX 语法：
```markdown
行内公式：$E=mc^2$
块级公式：$$E=mc^2$$
```
