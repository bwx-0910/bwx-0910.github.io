⚙️ GitHub 配置

首次使用需要配置
需要创建一个 GitHub Personal Access Token 才能在线编辑笔记。
点击这里生成 Token （需要勾选 repo 权限）

如何获取 Token？

  1. 访问 GitHub Settings → Tokens
  2. 点击 "Generate new token (classic)"
  3. 勾选 repo 权限
  4. 生成并复制 Token
  5. 粘贴到上面的输入框
