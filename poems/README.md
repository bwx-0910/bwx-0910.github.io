# 古诗词目录

存放古诗词摘录的 Markdown 文件。

## 文件格式

每个 `.md` 文件代表一首诗或一篇古文：

```markdown
---
title: 静夜思
author: 李白
dynasty: 唐
tags: 思乡, 月夜
date: 2026-01-16
---

床前明月光
疑是地上霜
举头望明月
低头思故乡
```

## 字段说明

- `title`: 诗词标题（必填）
- `author`: 作者（选填，默认"佚名"）
- `dynasty`: 朝代（选填）
- `tags`: 标签，逗号分隔（选填）
- `date`: 添加日期（自动生成）

## 正文格式

- 每行一句诗
- 空行会被忽略
- 支持长篇古文（自动换行）
